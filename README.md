# PAC-Autosetter

Applied form of the proof of concept from [Auto-HM-PAC-Gen](https://github.com/Bak-Tensu/Auto-HM-PAC-Gen).
An FF extension used to either :
 - Apply a PAC function (not file) automatically every 30 minutes according to a specified list, and on specified domains.
 - Use a specified `proxy:port` to apply a non-automatically renewed PAC function on the same specified domains.
